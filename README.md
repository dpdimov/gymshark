# Gymshark Financial Analysis Explorer

An interactive single-page tool for exploring Gymshark's financial performance from FY2015 to FY2021. Built as a standalone HTML file with no dependencies — just open `index.html` in a browser.

## Features

- **P&L** — Revenue, costs, and profitability trends with margin analysis
- **Balance Sheet** — Asset composition, funding structure, and growth
- **Returns** — DuPont decomposition of return on assets into profitability, turnover, and leverage
- **Working Capital** — Cash cycle days, working capital efficiency, and full cash flow statement
- **Levers** — Interactive scenario tool: adjust price, volume, COGS, overheads, and working capital to see impact on EBIT and cash flow

All charts are rendered on HTML5 canvas with no external charting libraries. Figures are in GBP thousands unless otherwise noted.

## Data

Source data from Gymshark's filed accounts (Companies House). Revenue grew from £8.7M (2015) to £437.6M (2021).

## Usage

Open `index.html` in any modern browser. No build step or server required.
